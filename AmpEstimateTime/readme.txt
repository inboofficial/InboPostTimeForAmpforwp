== Description ==
Inbo Post Time For ampforwp is a plugin for wordpress, and an extension for ampforwp plugin
to estimate and show the posts reading time

* Custom color and classes for estimated time box
* Custom text templates
* Custom icon html element
* Font-awesome and Flat-icon uicons support

1. clone the plugin using: clone https://github.com/inboofficial/InboPostTimeForAmpforwp
2. Zip AmpEstimateTime folder
3. In your website admin panel, go to Plugins -> 'add new' and click the â€˜upload pluginâ€™ button.
3. Click 'choose file' button in shown dialog and select the file in step one and click open button.

5. congregates you have installed the inbo posts time for ampforwp plugin
